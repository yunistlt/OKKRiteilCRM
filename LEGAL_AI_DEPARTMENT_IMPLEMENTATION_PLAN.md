- [ ] Завершить ingest pipeline документов:
  - [x] загрузка PDF/DOCX/TXT;

  - [ ] надежная обработка scan-heavy документов;

- [ ] Расширить rule engine для redlining:
  - [ ] красный флаг: неустойка выше 1%;

  - [ ] красный флаг: невыгодная подсудность;
  - [ ] красный флаг: передача прав на интеллектуальную собственность;
  - [ ] желтый флаг: автоматическая пролонгация;
  - [ ] желтый флаг: размытые сроки приемки;
  - [ ] желтый флаг: персональные данные без явного согласия.
- [ ] Добавить проверки обязательных разделов:
  - [ ] форс-мажор;

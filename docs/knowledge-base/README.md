# Knowledge Base (Voice of Customer)

**Статус:** 🟡 В разработке (Plan + Docs готовы)

## Описание

Централизованная база знаний, построенная на основе Voice of Customer (VoC) данных:

### Функции
- **Исторический майнинг** — извлечение вопросов из старых диалогов
- **Кластеризация** — группировка вопросов по интентам и частотности
- **Ручная модерация** — review и validation всех новых кластеров
- **Continuous updates** — добавление новых вопросов без даунтайма
- **FAQAccordion UI** — красивая интеграция в фронтенд

### Типы сущностей
- **Questions** — часто задаваемые вопросы от клиентов
- **Claims** — претензии и жалобы
- **Remarks** — замечания и предложения

## Процесс наполнения

1. **Mining:** scripts/voc_historical_mining.ts извлекает вопросы из старых диалогов
2. **Clustering:** скрипт группирует вопросы по семантике и подсчитывает частотность
3. **Moderation:** администратор смотрит новые кластеры через UI и утверждает их
4. **Publishing:** одобренные вопросы появляются в FAQAccordion на фронте
5. **Continuous:** новые вопросы добавляются по мере использования системы

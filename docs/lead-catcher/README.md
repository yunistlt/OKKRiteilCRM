# Lead Catcher (Елена - Продуктолог)

**Статус:** ✅ Реализован и работает в production

## Описание

Модуль для сбора лидов и контактов через AI-консультанта на сайте:

### Возможности
- **H2H консультация:** Елена отвечает на вопросы клиентов в реальном времени
- **Детекция контактов:** Автоматическое извлечение телефонов, email, Telegram из диалога
- **Прием файлов:** Загрузка спецификаций и ТЗ клиентом
- **Интеграция с ОКК:** Автоматическое создание заказов из собранных лидов
- **История диалогов:** Сохранение thread-ов для возврата к контексту

### Компоненты

| Компонент | Описание | Статус |
|-----------|---------|--------|
| Widget | Web widget на сайте клиента (мобильный + desktop) | ✅ Работает |
| Detect contacts | Извлечение телефонов, email, Telegram из текста | ✅ Работает |
| File upload | Загрузка и хранение файлов спецификаций | ✅ Работает |
| Lead creation | Автоматическое создание заказа в RetailCRM | ✅ Работает |
| Thread history | Сохранение и воспроизведение диалогов | ✅ Работает |
| Analytics | Метрики по сбору лидов в ОКК | ✅ Работает |

### Lead Catcher Hub

В разделе ОКК выделен специальный hub для мониторинга активности Елены:
- Все диалоги с виджета сохраняются в реальном времени
- Менеджеры видят, на каком этапе находится консультация
- Максим может оценить корректность ответов Елены
- Данные привязываются к заказам в RetailCRM

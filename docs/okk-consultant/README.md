# ОКК Консультант (Семён)

**Статус:** Production, в развитии  
**Ответственный:** Team ОКК

## Описание

Консультант ОКК (Семён) — это ИИ-ассистент, который объясняет:
- правила и критерии ОКК
- формулы расчета скоров (deal_score, script_score, total_score)
- источники данных и доказательства решений системы
- причины галочек и крестиков по критериям
- способы улучшения оценок

**Главный принцип:** Не просто справочник, а методолог-консультант, объясняющий смысл, а не только термины.

## Source of Truth

Для каждого класса знаний:
- **Section catalog** (code) → master source для экранных объяснений
- **Knowledge base** (DB) → для fallback и доп. контента
- **Prompt layer** (DB) → для правил ответа и style guidelines

## Архитектурные инварианты

🔒 **Security boundary** — Role-based ограничения применяются до попадания в промпт  
📝 **Единый источник контента** — Каждый класс знаний имеет определённый master-source  
🗺️ **Routing по структуре** — Опираемся на alias/entity/mode, не на free-text prose  
📊 **Full-by-default** — Полный ответ — стандарт, short только как follow-up  
🧪 **Regression first** — Любая правка сопровождается регрессией

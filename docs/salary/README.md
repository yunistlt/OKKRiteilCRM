# Раздел «Зарплата ОП» — вся документация

**Единая папка по зарплатному модулю.** Любая работа по ЗП — отсюда. Других мест с доками по зарплате нет.

| Документ | Тип | О чём |
|---|---|---|
| [OVERVIEW.md](OVERVIEW.md) | **Пояснение (as-built)** | ⭐ Начни отсюда. Как устроен модуль сейчас: принципы, поток данных, каталог блоков, БД, API, UI, рецепты. |
| [TASKS.md](TASKS.md) | **Задачи** | Открытый бэклог: что доделать, известные баги/долг, идеи. |
| [PLAN.md](PLAN.md) | **План** | План реализации по этапам. |
| [DECISIONS.md](DECISIONS.md) | **Пояснение (решения)** | Журнал принятых развилок и принципов (почему так). |
| [TZ.md](TZ.md) | **ТЗ** | Исходное техническое задание. |

**Правила раздела:**
- Все новые доки по зарплате класть **только сюда** (`docs/salary/`).
- Менять реализацию — сперва прочитать `OVERVIEW.md`; после изменений — обновить его.
- Дизайн/код — по [/golds](../../golds/). Принцип «ноль хардкода» — все числа в БД (effective-dated).

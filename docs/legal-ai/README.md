# Legal AI Department (Legal Agent Suite)

**Статус:** 🟡 4 спринта до production (Дарья → Лев → Борис → Григорий)

## Описание

Legal-направление добавляет в ОКК полнофункциональную юридическую консультацию и контроль с четырьмя специализированными агентами:

| Агент | Роль | Статус |
|-------|------|--------|
| **Дарья** | Legal Helpdesk (первая линия) | 🟡 В разработке |
| **Лев** | Contract Redlining (анализ договоров) | 🟡 MVP закрыто |
| **Борис** | Due Diligence (проверка контрагентов) | 🟡 MVP закрыто |
| **Григорий** | Претензионная работа (debt recovery) | 🔵 Планируется |

## Ключевые возможности

### Дарья — Legal Helpdesk
- Ответы на вопросы о возвратах, претензиях, NDA
- Intent routing и fallback на ручную эскалацию
- Генерация типовых документов из шаблонов
- Role-based sanitization (не раскрывает внутренние правила)

### Лев — Contract Redlining
- Загрузка PDF/DOCX контрактов
- Автоматический OCR для сканов
- Детекция red flags (неустойка > 1%, односторонние условия, невыгодная подсудность)
- Redlining report с предложениями для протокола разногласий

### Борис — Due Diligence
- Проверка контрагента по внешним источникам (Dadata, ФНС, реестры)
- Risk scoring (green/yellow/red)
- Автоматическое блокирование при красном риске
- Детальный breakdown по ролям доступа

### Григорий — Претензионная работа
- Детекция просрочки (> 14 дней)
- Автогенерация черновика претензии
- Расчёт пени и сбор документов
- Workflow передачи на визу юристу

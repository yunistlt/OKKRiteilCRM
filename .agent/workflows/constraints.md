# Constraints and Guidelines

- **Language Policy**: Entire user interface, AI-generated explanations, reasons, and details MUST be strictly in **RUSSIAN**.
- **No English placeholders**: Avoid using English terms like "outgoing", "incoming", "pending", etc. in user-facing text or violation details.
- **Rule Engine**: All new rules generated by AI must have Russian names and descriptions.
- **Semantic Analysis**: Reasoning and evidence summaries must be in Russian.

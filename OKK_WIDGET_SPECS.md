# Спецификация виджета ОКК-Консультант (Елена) — КЛАССИЧЕСКИЙ ДИЗАЙН

Этот документ является эталоном функциональности и ВНЕШНЕГО ВИДА виджета. ПРИ ЛЮБОМ ИЗМЕНЕНИИ КОДА все нижеперечисленные параметры ДОЛЖНЫ БЫТЬ СОХРАНЕНЫ.

## 1. Внешний вид (Design System)
- **Стиль**: Классический Webasyst (строгий, премиальный).
- **Размеры**: Ширина `360px`, Высота `550px`.
- **Позиция**: Справа, снизу. Отступ от низа `100px` (чтобы не перекрывать уведомления о куки/конфиденциальности).
- **Хедер**: Белый фон (`#ffffff`), высота `70px`, тонкая серая линия снизу (`#f0f0f0`).
- **Свернутое состояние**: Виджет просто съезжает вниз, оставляя видимым только хедер (70px). **НИКАКИХ круглых плавающих аватарок в свернутом виде.**
- **Закругление**: `24px` (только для основного контейнера).

## 2. Конфигурация (WIDGET_CONFIG)
- `autoExpandDelay`: 30000 мс (30 секунд) — время до автоматического разворота.
- `pollingInterval`: 3000 мс — частота опроса новых сообщений.
- `typingSpeed`: 30 мс — скорость анимации печатания текста.

## 3. Ключевые функции UI
- **Auto-Expand**: Автоматический разворот окна через 30 секунд для новых посетителей (если не было клика).
- **Preview Bubbles (Превью)**: Текстовое облачко `okk-chat-preview`, которое появляется НАД свернутым хедером (на высоте `100px` от низа), когда Елена присылает сообщение.
- **File Upload (Скрепка)**: Кнопка-скрепка для загрузки файлов. Обязательна поддержка статусов «Загрузка...» и визуального подтверждения в чате.

## 4. Интеллектуальные функции
- **Magic Greeting**: При инициализации (`init`) Елена должна упоминать товар из заголовка страницы (H1), если он есть.
- **Tracking**: Передача UTM-меток, истории страниц и товаров в каждом `apiCall`.
- **Automatic Callback**: Бэкенд-детекция номера телефона и запуск звонка через Telphin.

## 5. Правила сохранения кода
- **CSS**: Стили должны быть компактными, но читаемыми. Не менять `z-index: 2147483647`.
- **JS**: Не удалять проверку `okk_has_interacted` для авто-разворота.
- **DOM**: Не менять ID элементов (`okk-chat-widget`, `okk-chat-preview`, `okk-chat-header`).
